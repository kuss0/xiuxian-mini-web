# Xiuxian Mini Web

Telegram 修仙 CHAT 群组辅助图形界面。

这个项目不是替代游戏本体，也不是自动挂机脚本。游戏本体仍然在 Telegram 群里：玩家聊天、发送指令、接收韩天尊回复。Mini Web 的目标是把高噪声聊天流整理成更清爽的游戏辅助界面，让玩家更容易看懂消息、筛选事件、复制或确认发送指令，并管理 Telegram 官方定时消息。

## 核心边界

- 只做 Web 版，暂不做移动 App 接入。
- 只读 Telegram 群消息并做结构化展示。
- 指令出口只允许两类：
  - 用户确认后的手动发送。
  - Telegram 官方定时消息。
- 不做自动连发链路。
- 不做自动重试、自动补发、自动刷状态。
- 不把游戏群变成脚本日志输出场。

## 主要体验

- 像网络游戏聊天窗口一样，把世界、系统、副本、资源、风险、我的相关消息分流展示。
- 支持单独查看频道，也支持勾选合并展示。
- 面向用户时优先使用场景视图：
  - 我的修仙
  - 游戏大厅
  - 事件中心
  - 操作台
- 消息卡片默认摘要展示，原始 Telegram 文本可展开。
- UI 可以使用繁体中文标签，但发送给游戏 bot 的命令保持游戏要求的简体格式。

## 初始目录
